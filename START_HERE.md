## 📞 NEED HELP?

### During Deployment
1. Check **STEP_BY_STEP_DEPLOYMENT.md** for detailed steps
2. Check **DEPLOYMENT_GUIDE.md** for troubleshooting
3. Check service logs in Render/Vercel dashboards
4. Check browser console (F12) for errors

### Common Issues

**"Backend not responding"**
→ Wait 10 full minutes for first deployment
→ Check Render logs for errors

**"Frontend shows API error"**
→ Check env-config.js has correct backend URL
→ Check CORS setting in Render

**"Can't login to admin"**
→ Check username is exactly "admin"
→ Check password in Render environment variables
