## 📁 Structure

```
portfolio/
├── frontend/          # Public portfolio website
│   ├── index.html    # Main portfolio page
│   ├── css/          # Stylesheets
│   └── js/           # JavaScript files
└── backend/          # Backend API (private)
    └── src/          # Spring Boot application
```

## 🛠️ Technologies Used

### Frontend
- HTML5, CSS3, JavaScript
- Responsive Design

### Backend
- Java Spring Boot
- Maven

## 📦 Deployment

### Frontend (GitHub Pages / Netlify / Vercel)
1. Deploy only the `frontend` folder
2. Set `index.html` as the entry point
3. Configure build settings as needed

### Backend
- Deploy separately on a server with Java 17+
- Configure CORS to allow frontend access
- Set up environment variables for security

## 🔒 Security Notes

- Admin dashboard is kept private and not included in public repository
- User uploads and profile data are excluded from version control
- Backend JDK and build artifacts are gitignored
